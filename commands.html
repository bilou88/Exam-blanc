Commandes Git essentielles

Liste de commandes Git

  • git init
  • git clone
  • git status
  • git add
  • git commit
  • git push
  • git pull

Tableau : Commande / Description

Commande Description
git init Initialise un nouveau dépôt Git dans le répertoire courant.
git clone <url> Copie un dépôt Git distant sur la machine locale.
git status Affiche l'état des fichiers (modifiés, suivis, non suivis).
git add <fichier> Ajoute un ou plusieurs fichiers à l'index (zone de staging).
git commit -m "message" Enregistre les changements de l'index dans l'historique du dépôt.
git push Envoie les commits locaux vers un dépôt distant.
git pull Récupère et fusionne les changements du dépôt distant.
