Commandes Git essentielles

Liste de commandes Git

  • git init
  • git clone
  • git status
  • git add
  • git commit
  • git push
  • git pull

Tableau : Commande / Description

Commande Description
git init Initialise un nouveau dépôt Git.
git clone <url> Copie un dépôt Git distant sur la machine locale.
git status Affiche l'état des fichiers.
git add <fichier> Ajoute les fichiers modifiés à la zone de préparation en vue du prochain commit.
git commit -m "message" Enregistre dans le dépôt les fichiers qui ont été ajoutés à la zone de préparation.
git push Envoie les commits locaux vers un dépôt distant.
git pull Récupère et fusionne les changements du dépôt distant.
